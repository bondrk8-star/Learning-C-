# Мои учебные проекты на C#
Этот репозиторий создан в рамках моего самостоятельного изучения C#.
Здесь собраны небольшие учебные проекты , которые демонстрируют мое понимание основ языка.

## Содержание

### Базовый синтаксис
- CalculatorApp - работа с консолью
- CalculatorTrue - простое консольное приложение-калькулятор
- EvenOrOdd - условие if-else, определяющее число в четные или нечетные
- GuessTheWord - условие if-else для "отгадай слово"
- TernaryOperator - позитивное или отрицательное число с помощью тернарного оператора
- CountEvenAndOdd - цикл while для подсчета количества четных и нечетных цифр в числе
- CircleForPractice - цикл for для подсчета суммы цифр в числе

### Массивы и методы
- Massive - изучение массивов, в том числе многомерных и зубчатых
- Methods - создание статических методов
- References - изучение ref, передача параметров по ссылке

### Проекты
- Triangles - рисование треугольников с помощью циклов for
- TrianglesNew - также треугольник с использованием for и SetCursorPosition

---

## Технологии
- C#, .NET 10
- Консольный ввод-вывод
- Работа с массивами и методами

---

## Как запустить
Откройте решение в VS, выберите нужный проект в обозревателе решений и нажмите "Ctrl+F5" для запуска без отладки.
